# Samples

This directory stores small, versioned inputs that keep notebooks, tests, and CLI workflows
working against the same examples.

## Contents

- `training/repository_training_examples.yaml`: starter question-answer pairs for DSPy-oriented experiments.
- `population/repository_population_candidates.yaml`: prioritized corpus candidates for staged repository ingestion.
- `logs/`: post-push GitHub Actions inspection logs captured with `gh`.
