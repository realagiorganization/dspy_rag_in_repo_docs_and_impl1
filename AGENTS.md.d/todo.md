# TODO Instructions

Treat `TODO.md` as part of the normal request workflow.

Before starting work:

- Read `TODO.md`.
- Fold any immediately relevant cleanup item into the current task.

Before finishing work:

- Remove TODO items that were completed.
- Rewrite TODO items that changed scope so they stay concrete and actionable.
- Add new TODO items only when they represent real unfinished follow-up, not vague possibilities.
